# PDetalles del ESP32

Para este interactivo se requiere un sensor 49e 945bc conectado a 3.3V, GND y el pin de señal en el pin 4 del ESP32. Para el botón se requiere uno normalmente abierto conectado entre los pines GND y 23.

- El programa `01_Sensor_Tester` es para realizar la prueba del sensor
- El programa `02_Sensor_y_Boton`es para probar el botón
